## Selected Work

**VoiceBridge** — 🏆 1st Place, Honeywell Designathon 2026
A 5-stage emotion-preserving accent-conversion pipeline, translating Indian-accented speech into American or British English without losing the speaker's tone. Built on faster-whisper, Wav2Vec2, ECAPA-TDNN, and OpenVoice v2, with a FastAPI backend streaming live logs to a React dashboard over SSE.
`Python` `FastAPI` `React` `OpenVoice v2`

**Multi-Agent Shopify Workflow Automation**
A tri-agent LangChain system that evaluates SKU profitability and runs ML-driven stockout risk, closing the loop with an n8n workflow that triggers automated repricing through the Shopify Admin API.
